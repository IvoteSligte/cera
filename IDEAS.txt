implicit function parameters:
    ```
    use (n: int) {
        func square(): int {
             // uses the `n` from the outer scope
             return n * n;
        }

        func cube(): int {
             // implicitly forwards `n` to `square`
             return square() * n;
        }

        func plus_one_square(): int {
             // overrides the value of `n` passed to `square`
             return square(n=n+1);
        }
    }

    // functions outside of the `use` block must pass `n` explicitly
    square(100);
    ```

    This feature would practically replace objects as in OOP.
    A parser in C++:
    ```cpp
    class Parser {
    public:
        Parser(Stream stream) : stream(stream) {}

        void skip() {
            stream.skip();
        }

        ParseTree parse() {
            skip();
            // other logic
        }
    private:
        Stream stream;
    };

    Stream stream("file.txt");
    Parser parser(stream);
    auto output = parser.parse();
    ```

    A parser in Cera:
    ```
    use (stream: Stream) {
        func parse() {
            skip();
            // other logic
        }

        func skip() {
            stream.skip();
        }
    }

    stream := file_stream("file.txt");
    auto output = parse(stream);
    ```

loops as expressions:
    if-statements as expressions are somewhat common nowadays
    `result := if condition { "positive" } else { "negative" };`

    but having loops produce output similarly to the functional `map` operator can be useful as well
    ```
    elements := .[10, 20, 30];
    squared := for element in elements {
        element * element
    };
    ```
    preserving the container type by default, but making it possible to change the type using an explicit annotation
    maybe using a similar mechanism to Rust's IntoIterator/FromIterator
    ```
    pairs = .[.{"key", value}]
    map: HashMap := for pair in pairs { do_stuff(pair) }
    ```
    it is particularly useful to be able to skip elements that are not desired and end the loop at will
    ```
    evens := for n in .[0, 1, 2, 3] {
        if n % 2 != 0 {
           continue;
        }
        n
    }
    ```
